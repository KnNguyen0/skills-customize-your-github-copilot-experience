# 📘 Assignment: Python Testing and Debugging

## 🎯 Objective

Learn how to write unit tests and debug Python programs using `pytest` and structured assertions to make code more reliable.

## 📝 Tasks

### 🛠️ Write Unit Tests

#### Description
Create tests for existing Python functions to verify that they behave correctly for valid and invalid inputs.

#### Requirements
Completed assignment should:

- Install and use `pytest` to run tests.
- Create at least three test cases covering normal behavior and edge cases.
- Include tests for both valid input and error handling.
- Use `assert` statements to verify expected outcomes.

### 🛠️ Debug and Fix Code

#### Description
Inspect the starter code, identify bugs or incorrect behavior, and fix the functions so they pass your tests.

#### Requirements
Completed assignment should:

- Identify at least one logic bug in the starter code.
- Correct the code so all tests pass.
- Add comments explaining the bug and the fix.
- Run `pytest` and confirm the test suite completes successfully.
